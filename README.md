# Projeto de Classificação de Idade em C#

Bem-vindo ao **Projeto de Classificação de Idade em C#**. 


Este é um projeto simples em C# que demonstra como classificar uma pessoa em diferentes grupos etários com base na idade fornecida.

## Objetivo

O objetivo deste projeto é criar um aplicativo console que solicita ao usuário que insira sua idade e, com base na idade informada, classifica a pessoa em um dos seguintes grupos:

- **Infantil**: Idade entre 0 e 12 anos.
- **Adolescente**: Idade entre 13 e 17 anos.
- **Adulto**: Idade entre 18 e 64 anos.
- **Idoso**: Idade de 65 anos ou mais.

O projeto usa uma classe `Person` para armazenar a idade e determinar o grupo etário correspondente.

## Funcionalidades

- **Entrada de Idade**: Solicita ao usuário que insira sua idade.
- **Validação de Entrada**: Verifica se a entrada é um número inteiro válido.
- **Classificação Etária**: Classifica a pessoa em um grupo etário com base na idade informada.

## Estrutura do Projeto

- **Program.cs**: Contém a classe `Program` com o método `Main`, que gerencia a entrada do usuário e exibe a saída.
- **Person.cs**: Contém a classe `Person` com a propriedade `Age` e o método `GetAgeGroup` para determinar o grupo etário.
